# Chatbot Testing Instructions

This project contains multiple chatbot test scripts. Follow the instructions below to set up and run the tests.

## 1. Install Dependencies

Before running any test scripts, install the required Node.js packages:

```
npm install
```

## 2. Running Test Scripts

You can run any of the chatbot test scripts using Node.js. Use the following commands as examples:

- For Daals chatbot:
  ```
  node ./chatbot_test_daals.js
  ```
- For Daals UAT chatbot:
  ```
  node ./chatbot_test_daals_uat.js
  ```
- For Coco chatbot:
  ```
  node ./chatbot_test_coco.js
  ```
- For Coco UAT chatbot:
  ```
  node ./chatbot_test_coco_uat.js
  ```
- For Chinti chatbot:
  ```
  node ./chatbot_test_chinti.js
  ```
- For Chinti UAT chatbot:
  ```
  node ./chatbot_test_chinti_uat.js
  ```
- For Hook chatbot:
  ```
  node ./chatbot_test_hook.js
  ```
- For Hook UAT chatbot:
  ```
  node ./chatbot_test_hook_uat.js
  ```

Replace the script name as needed for your testing.

## 3. Notes

- Ensure you have Node.js installed on your system.
- If you add new test scripts, follow the same pattern to run them: `node ./<script_name>.js`
